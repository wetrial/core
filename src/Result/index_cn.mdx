## API
| 参数                 | 说明                                             | 类型    | 默认值 |
| -------------------- | ------------------------------------------------ | ------- | ------ |
| type|成功或失败'success'/'error'|string|-|
|title|标题|React.ReactNode|-|
|description|结果描述|React.ReactNode|-|
|extra|其他补充信息，自带灰底效果|React.ReactNode|-|
|actions|操作建议，一般放置按钮组|React.ReactNode|-|
